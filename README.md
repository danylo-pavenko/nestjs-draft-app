## NestJS App 
- Example app where contains auto and storae user data, autogetn docs by Swagger.

## Description
- App contains connect with PostgresQL database, where contains different configuration of environement. Using a Docker to containerization example.
- [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
